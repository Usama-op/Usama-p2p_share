# WebRTC File Share

Peer to peer file sharing using WebRTC, Firebase, and TailwindCSS.

## Deployment Instructions

1.  **Firebase Setup**:
    - Create a project in [Firebase Console](https://console.firebase.google.com/).
    - Add a Web App to get your configuration.
    - Enable **Cloud Firestore** and use the rules provided in `firestore.rules`.
2.  **Configuration**:
    - Replace the placeholder values in `firebase-config.js` with your Firebase credentials.
3.  **GitHub Pages**:
    - Push this code to a GitHub repository.
    - Enable GitHub Pages in the repository settings.
